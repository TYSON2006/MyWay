# My Way 6_T

## Description
My Way 6_T est une plateforme d'orientation et de conseil en carrière dédiée aux élèves de la classe de 6eme à la Terminale. Le projet vise à accompagner les étudiants dans la découverte de leurs passions et le choix de leur futur métier à travers une interface intuitive et des ressources adaptées au système éducatif.

## Objectifs du Projet
* Fournir une orientation scolaire claire et accessible.
* Proposer des fiches métiers détaillées (Génie Civil, Informatique, etc.).
* Aider à la transition entre le cycle secondaire et les grandes écoles.
* Centraliser les informations sur les filières d'excellence en Côte d'Ivoire.

## Technologies Utilisées
* HTML5 : Structure des pages web.
* CSS3 : Mise en page responsive (Mobile, Tablette, TV) et animations.
* C / Python : Algorithmes et logique de traitement de données.
* Git : Gestion des versions du projet.

## Installation et Utilisation
1. Cloner le dépôt : git clone https://github.com/votre-utilisateur/my-way-6t.git
2. Ouvrir le fichier index.html dans un navigateur.
3. Consulter les sections d'orientation par niveau scolaire.
